How to run:
    1: navigate to assignments/dist and open index.html in a web browser

How to modify:
    1: ensure ruby is isntalled
    2: ensure node is isntalled
    3: ensure bootstrap is installed
    4: install grunt file
        1: npm install -g grunt-cli
        2: npm install grunt --save-dev
        3: npm install grunt-contrib-sass --save-dev
        4: npm install grunt-contrib-uglify --save-dev
        5: npm install grunt-contrib-copy --save-dev
        6: npm install grunt-contrib-concat --save-dev
        7: npm install grunt-contrib-watch --save-dev
    5: Modify files in src with grunt running to update dist files.
